# CS50 
是哈佛大學開設的程式通識課程，內容包山包海，從 Scratch 教到 C 語言、演算法、資料結構、HTTP、Python 甚至是機器學習都有。
課程裡有蠻多上課有教到的東西，例如Heapsort、Linked List等等
